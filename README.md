# Bank Management System

A Java Swing + MySQL based ATM Banking System project.

## Features
- User Login
- Deposit Money
- Cash Withdrawal
- Fast Cash
- Mini Statement
- Balance Enquiry
- PIN Change

## Technologies Used
- Java
- Swing
- JDBC
- MySQL

## Tools Used
- VS Code
- MySQL Workbench

## How to Run
1. Clone the repository
2. Import MySQL database
3. Add required JAR files
4. Run Login.java
